项目说明
=======

本仓库包含若干与大数据分析实践相关的前端和后端项目。下面简要说明各个顶层目录的用途：

- ddtwk.site/
	- 说明：已编译（构建）后的前端静态资源目录，包含页面、JS、CSS、图片等。
	- 用途：直接部署到静态托管或作为前端静态文件集与后端分离部署。

- DjangoProject/
	- 说明：系统的“外部”项目（外部-facing），负责对外提供接口或面向外部用户的功能模块。
	- 用途：处理与外部交互、用户注册/登录、公开API（针对外网的搜索等）等。

- DjangoProject_internal/
	- 说明：系统的“内部”项目（内部服务/管理端），包含内部数据管理上传修改等、后台任务、内部API与人员管理功能。
	- 用途：处理网站内部人员的操作、与内部数据库、数据处理流程相关的实现。

- mysite/
	- 说明：单独实现进化树功能的项目目录。
	- 用途：实现与进化树构建、展示与相关分析逻辑有关的后端与前端代码。
